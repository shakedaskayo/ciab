# ciab config

Configuration management.

## show

Display the current configuration.

```bash
ciab config show [--config <path>]
```

## validate

Validate a configuration file.

```bash
ciab config validate [--config <path>]
```

## init

Generate a default configuration file.

```bash
ciab config init [--output <path>]
```

Creates `config.toml` with default values and comments.
